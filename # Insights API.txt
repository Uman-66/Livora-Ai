# Insights API

## Endpoint

POST /insights

### Request Fields

| Field | Type |
| user_id | integer |

### Response

Success:

```json
{
    "success": true,
    "analysis": {
        "overall_health_score": "integer",
        "flags": {
            "ast": "string",
            "alt": "string",
            "bilirubin": "string",
            "albumin": "string"
        },
        "apri_fib4_interpretation": "string",
        "ai_insights": [
            "string"
        ],
        "ai_summary": "string"
    }
}


### Notes

- Generates the final AI liver health assessment.
- Uses previously uploaded report data, calculated fibrosis scores, and ultrasound prediction.
- Returns a structured analysis object ready for display in the dashboard.
